Plane Social Content für 4 Plattformen.

Erstelle, sortiere und verwalte Beiträge für Facebook, Instagram, Pinterest und TikTok. Deine Planung bleibt lokal auf diesem Gerät und wird automatisch in LocalStorage gespeichert.

Anweisungen für Claude Code

  1. Diese Datei als lokale Single-Page-App öffnen oder mit einem statischen Server starten.
  2. Neue Content-Ideen im Formular erfassen und mit Plattform, Datum, Uhrzeit, Format und Status versehen.
  3. Über Filter nach Plattform oder Status fokussieren; Änderungen werden ohne Backend gespeichert.
  4. Mit dem JSON-Export lässt sich die Planung sichern oder an andere Tools übergeben.

Beitrag planen

Bilder werden als Daten-URL lokal im Browser gespeichert. Für viele oder sehr große Bilder empfiehlt sich der JSON-Export als Backup.

Hinweis: Die Daten werden nur in diesem Browser gespeichert. Nutze den Export, wenn du ein Backup brauchst.
